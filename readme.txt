Dynamic flexible ajax search widget

== Description ==

Costumizable search widget. Shows the search results dynamic on your sidebar.

Features:

* Show or don't show a searchfield or configure another input field
* Configure the search: post type, categories, tags, post title and content

== Installation ==

1. Upload the plugin to your `/wp-content/plugins/` directory and activate it
2. Drag the widget to your sidebar and configure the search settings there
